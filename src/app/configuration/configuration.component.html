Diese Applikation ermöglicht einen full data access. Der gesamte Datenbestand der Applikation (inkl. der Geodaten) kann jederzeit komfortabel über diese Seite heruntergeladen und migriert werden. Die Daten werden im standardisierten, Web-nativen JSON-Format bereitgestellt.

Achtung, der Gesamtdownload beinhaltet auch personenbezogene Daten der registrierten Benutzer dieser Applikation. Passwörter sind im Download jedoch nicht enthalten.

Grundsätzlich wird die Interoperabilität dieser Applikation sichergestellt über die Bereitstellung von standardisierten Service-Schnittstellen auf Basis von Open API. Die technische Dokumentation der Services wird auf dieser Webseite live bereitgestellt:
